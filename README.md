# Netflix Data Analysis Project

## Project Overview
This project analyzes Netflix movies and TV shows data using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Movies are more common than TV Shows
- USA produces most Netflix content
- Drama is the most popular genre
- Netflix content increased rapidly after 2015

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend Analysis

## Dataset
Netflix Movies and TV Shows Dataset
